# ORA Fashion Streamlit App

This app provides an interactive customer intelligence dashboard for ORA Fashion.

## Features
- Upload Customers and Orders Excel files
- View executive KPIs
- Analyze revenue trend
- Explore customer demographics
- Review CLV distribution
- Identify churn risk groups
- Download customer intelligence output

## Run Locally

```bash
pip install -r requirements_streamlit.txt
streamlit run app/streamlit_app.py
```
